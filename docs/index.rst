Emelem is a machine learning model platform designed to be intuitive
and interactive.


----------------

Overview
=======================================

Features
---------

- Create and share machine learning models
- An extensible, high-granularity security/permission model allowing
  intricate rules on who can access individual features and the models
- Enterprise-ready authentication with integration with major authentication
  providers (database, OpenID, LDAP, OAuth & REMOTE_USER through
  Flask AppBuilder)
